# LIBSAX2
> Browser Sax2 driver for XML pull parser implementation (hjs-xmlpull and hjs-kxml2).

SAX2 classes: InputSource, DefaultHandler, SaxDriver.

## Installation

Node:

```sh
npm install libsax2 --save
```
